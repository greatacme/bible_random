# 오늘의 성경 말씀

주제별 성경 말씀을 랜덤으로 보여주는 단일 페이지 웹앱입니다.

**Live**: https://greatacme.github.io/bible_random/

## 기능

- 4개 주제 카테고리에서 랜덤 말씀 표시
  - **위로가 필요할 때** — 시편 23편, 이사야 41:10, 마태복음 11:28 등 21구절
  - **소망 / 사랑 / 믿음** — 고린도전서 13장, 히브리서 11:1, 요한1서 4:8 등 20구절
  - **지혜 / 용기** — 여호수아 1:9, 잠언 시리즈, 야고보서 1:5 등 19구절
  - **감사 / 찬양** — 시편 100, 103, 150편, 데살로니가전서 5:16-18 등 18구절
- 카테고리 버튼 클릭 시마다 새로운 말씀 표시
- 말씀 + 장절 정보 클립보드 복사 기능

## 기술 스택

- 단일 `index.html` 파일 (외부 의존성 없음)
- [bible.helloao.org](https://bible.helloao.org) API — 개역한글 성경 (`kor_old`)
- API 키 불필요
- GitHub Pages 호스팅

## 성경 번역

**개역한글 (Korean Revised Version, 1961)** — 퍼블릭 도메인

현대 한국어 번역 (개역개정, 새번역 등)은 대한성서공회 저작권으로 인해 무료 API에서 제공되지 않습니다.
