# 오늘의 성경 말씀

페이지를 열 때마다 엄선된 78구절 중 하나를 랜덤으로 보여주는 단일 페이지 웹앱입니다.

**Live**: https://greatacme.github.io/bible_random/

## 기능

- 3초 인트로 애니메이션 (십자가 + 확장 링 + 글로우 효과)
- 4개 주제에서 엄선된 총 78구절 중 랜덤 표시
  - **위로가 필요할 때** — 시편 23:4, 이사야 41:10, 마태복음 11:28 등 21구절
  - **소망 / 사랑 / 믿음** — 요한복음 3:16, 고린도전서 13:13, 히브리서 11:1 등 20구절
  - **지혜 / 용기** — 여호수아 1:9, 잠언 1:7, 야고보서 1:5 등 19구절
  - **감사 / 찬양** — 시편 100:4, 시편 150:6, 데살로니가전서 5:18 등 18구절
- 말씀 등장 시 폭죽/컨페티 효과
- 말씀 + 장절 정보 클립보드 복사
- 새로고침으로 새 말씀 보기

## 기술 스택

- 단일 `index.html` 파일 (HTML + CSS + JS)
- [bible.helloao.org](https://bible.helloao.org) API — 개역한글 성경 (`kor_old`)
- Google Fonts (Noto Serif KR)
- API 키 불필요
- GitHub Pages 호스팅

## 성경 번역

**개역한글 (Korean Revised Version, 1961)** — 퍼블릭 도메인

현대 한국어 번역 (개역개정, 새번역 등)은 대한성서공회 저작권으로 인해 무료 API에서 제공되지 않습니다.
